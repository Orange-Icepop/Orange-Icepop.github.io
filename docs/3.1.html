Chapter3.1 MC购买与安装

本文介绍的是Minecraft国际版的购买与安装方法，你可能是因为厌烦了网易版各种恶心的操作或者想要加模组才来看这个文档的，不用担心，包在我身上！

3.1.1 购买

撰写本文时（2024年初），在中国大陆地区已经无法通过登录官网购买MC国际版，因此我们需要从官方启动器购买。

提示

MC国际版主要分为两个版本：基岩版和Java版。根据官网上的信息，Mac和Linux仅支持Java版，PC则两个版本都可以使用，其余的平台仅支持基岩版（Android和iOS实际上可以使用特别的启动器运行Java版）。基岩版是除了Java版外所有版本的统称。各个平台的购买不通用。

注意

如果使用第三方启动器（如PCL2），可以在不购买的情况下游玩MC Java版，但将无法加入开启了正版验证的服务器！

购买时，您需要一台安装了Microsoft Store的Windows PC，一个Microsoft账户，与在线支付方式（微信与支付宝均可）

打开Microsoft Store，查找Minecraft，选择Minecraft启动器并下载安装。

打开Minecraft启动器，登录Microsoft账户，点击Java Edition页面下的购买按钮，按照步骤操作，最终付款成功即可。

3.1.2 安装

如果你要游玩Minecraft基岩版，在官方启动器的Minecraft for Windows界面下下载最新版本，打开游玩即可。

Minecraft Java版的技术门槛相对较高，但它的Mod（模组）与整合包数量远超基岩版，可选择性更多。鉴于多数国际版玩家均选择Java版，建议各位使用Java版。

在安装Java版MC前，请先从甲骨文官网下载对应版本的Java，对照表如下（数据来自cv35088159）：

Minecraft 版本 推荐 Java 版本 最低 Java 版本
最早 ~ 1.11.2 Java 8(LTS) 未知
1.12 ~ 1.16.5 Java 8(LTS) Java 8(LTS)
1.17 ~ 1.17.1 Java 17(LTS) 或 Java 18 Java 16
1.18 ~ 1.20.4 Java 18 或 Java 21(LTS) Java 17
1.20.5 ~ 最新 Java 21(LTS) Java 21(LTS)且操作系统必须为64位

安装完毕后，您就可以打开启动器开始安装MC了。下方将以4种不同的主要启动器示例，若您的电脑性能不足，可以查看优化指南。

官方启动器：在Java Edition页面点击配置页面，新建配置，选择你想安装的版本。启动器将自己开始下载至默认安装位置。

官启仅支持原版核心（核心代号Vanilla）的安装。若想安装Forge或Fabric核心的MC用于安装模组，请使用另外3种启动器或查看手动安装教程。

PCL2：点击下载栏，从左侧栏选择游戏下载。选择想要安装的版本，在下一个页面选择想要添加的模组加载器和优化器，编辑好名称，点击下载即可。

HMCL：点击主页左侧中央的下载，选择版本，选择模组加载器和优化器，点击安装即可。

BakaXL：转到开始页，点击笨蛋广场，在右上角找到游戏下载。点击你想用的版本，选择需要的模组加载器，确定安装即可。

如果使用PCL2，建议您在启动器的设置中打开版本隔离，选择隔离所有版本，从而分清各个版本并防止模组混乱。

强烈建议您使用后面3种启动器。关于第三方启动器的优势与下载方式可以转至下一章节。下载及官网在下一章节第二小节（3.2.2）末尾。
